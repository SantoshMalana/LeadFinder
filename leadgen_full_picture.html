1
Reddit scraper — fetch new posts
Runs every 6 hours via Inngest cron. Fetches top 100 newest posts from target subreddits using Snoowrap.
const r = new Snoowrap({ userAgent, clientId, clientSecret, username, password })
const posts = await r.getSubreddit('forhire').getNew({ limit: 100 })
2
Deduplication via Redis
Before scoring anything, check if we've seen this post ID before. Skip if yes, mark if no.
const key = `seen:${post.id}`
const exists = await redis.get(key)
if (exists) continue
await redis.set(key, '1', { ex: 2592000 }) // 30 days TTL
3
Keyword pre-filter (cheap, no AI)
Simple string check before calling the LLM. Cuts ~80% of posts instantly at zero cost.
const text = (post.title + post.body).toLowerCase()
const hit = campaign.keywords.some(kw => text.includes(kw))
if (!hit) continue
4
AI scoring with Groq (llama-3.3-70b)
Only survivors from step 3 reach here. Score 1–10 for hiring intent. Response is JSON.
const res = await groq.chat.completions.create({
  model: 'llama-3.3-70b-versatile',
  messages: [{ role: 'user', content: scoringPrompt }],
  response_format: { type: 'json_object' }
})
const { score, reason } = JSON.parse(res.choices[0].message.content)
5
Save to Supabase if score ≥ 7
Low-score posts are discarded. High-score leads are stored and appear in the dashboard instantly via Supabase Realtime.
if (score >= 7) {
  await supabase.from('leads').insert({
    campaign_id, platform: 'reddit', post_id: post.id,
    post_title: post.title, post_url: post.url,
    score, score_reason: reason, status: 'new'
  })
}
6
Alert user if score ≥ 9
Top-tier leads trigger an immediate email via Resend. Later: Slack webhook.
if (score >= 9) {
  await resend.emails.send({
    from: 'leads@yourapp.com',
    to: user.email,
    subject: `🔥 ${score}/10 lead found: ${post.title.slice(0,50)}`,
    html: emailTemplate(post, score, reason)
  })
}
7
User clicks "Generate Reply"
Calls /api/reply with the lead data + user's voice profile. Gemini Flash writes the DM in ~1 second.
const reply = await gemini.generateContent(
  buildReplyPrompt(post, user.voice_profile, user.portfolio_summary)
)
return reply.response.text() // ready to copy & send
